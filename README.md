# Hondo Sushi Bath — concept preview

Mobile-first one-page concept preview for Hondo Sushi in Bath.

- This is not the restaurant's official website.
- Search engines should not index it.
- The preview uses lightly refined food, interior and brand imagery from Hondo Sushi's current official website; sources and refinements are recorded in `image-sources.md`.
- Business information is based on public sources and must be confirmed before official production use.
- No customer reviews, Google ratings or unverified awards are used.

Static deployment: publish the repository root with no build command.
